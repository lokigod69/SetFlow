# SETFLOW — Roadmap

## In v1 (spec §11, marked ●)
- Transition cheat notes on every pair (key relation, BPM delta, blend style) — the teaching feature.
- Set sheet: printable one-pager (arc graph + tracklist + per-transition cheat notes).
- Swap-a-track with neighbor-aware alternatives (long-press a node → 3 compatible swaps).
- Set history + "remix this set" (start from a saved set, mutate the intent).
- Camelot mini-wheel on track cards.

## Parked (spec §11, marked ○ — do not build in v1)
- **Practice log** — after a session, mark which transitions worked/failed; the brain learns the user's taste from feedback and biases future sets.
- **Double-drop finder** — pairs in the pool that could layer: same key, compatible BPM, complementary energy.
- **Tempo-journey templates** — warm-up / peak-time / sunset / after-hours presets that pre-shape the energy curve.
- **TEMENOS-style export** — "emotional intent → realized set" pairing exported to the user's other apps.

## Ideas that surfaced during the build (park here, don't scope-creep v1)
- **Ditter graph integration** — the user's own graphing widget as a second ArcRenderer adapter (interface is ready; widget was not attached at kickoff).
- **Rekordbox XML export** — spec lists it as nice-to-have behind Advanced; ship if time allows within v1, else it lives here.
